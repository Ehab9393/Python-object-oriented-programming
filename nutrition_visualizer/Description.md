Description:

A Python program that tracks user dietary intake, calculates BMI, and visualizes nutrition data using multiple chart types. Users can input their personal details and daily food servings, then generate BMI charts, bar charts, line charts, and pie charts comparing actual vs recommended intake. The program also allows exporting user data to a text file for record-keeping.

Key Features:

Calculates BMI from user weight and height.

Stores daily food intake and compares it to recommended servings.

Implements multiple chart strategies: BMI chart, bar chart, line chart, and pie chart.

Uses Strategy Design Pattern for flexible visualization.

Saves charts as images and optionally exports user data to a text file.

Provides clear visual comparison of actual vs recommended intake across food groups.
