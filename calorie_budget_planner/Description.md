Description:

An interactive Python program that helps users track daily food intake against a personalized calorie goal. Users can log multiple meals, input calories and macronutrients (protein, carbs, fats), and receive a daily summary showing total intake and remaining calories. The program also allows exporting the daily meal log to a text file for record-keeping.
